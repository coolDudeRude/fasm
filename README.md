# xSVM Assembler

## Setup

A python virtual environment with [parsy](https://github.com/python-parsy/parsy) is required.
```
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Usage
Run `fasm.py` with `--help` flag to see usage information.
```
./fasm.py --help
```
